# -Cab-Booking-Service

Cab Booking Service is a full-stack web application that allows users to book rides online in a simple and efficient way. The system provides real-time booking functionality with a user-friendly interface.

This project is built using:

Node.js
Express.js
MongoDB
Angular
Socket.io

🔥 Features

User registration and login
Cab booking request system
Real-time updates using Socket.io
Admin dashboard (optional)
Responsive web interface
REST API integration

🛠️ Technologies Used

Backend: Node.js + Express
Frontend: Angular
Database: MongoDB
Real-time communication: Socket.io
Package Manager: npm

▶️ How to Run the Project

Install dependencies

npm install

Start MongoDB server

Build Angular app

npm run build

Start the server

npm start

Open in browser

http://localhost:3000

📌 Purpose of Project

This project is created to demonstrate full-stack development skills including backend API development, frontend integration, real-time communication, and database management.
